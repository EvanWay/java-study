## 项目概述

这是一个简单的贪吃蛇游戏项目，使用纯 HTML/CSS/JavaScript 实现，单文件自包含。

## 文件结构

- `snake-game.html` - 完整的贪吃蛇游戏，包含所有 HTML、CSS 和 JavaScript

## 运行方式

直接在浏览器中打开 `snake-game.html` 即可运行：

```bash
open snake-game.html
```

## 游戏说明

- 方向键控制蛇的移动
- 空格键开始/暂停游戏
- 得分使用 localStorage 持久化

# 4个月宝宝乘坐高铁指南

> 整理日期：2026-06-28  
> 适用情况：4个月宝宝与成年人一起乘坐高铁，宝宝尚未办理居民身份证。

## 先看结论

- **户口本原件可以作为宝宝乘车的有效身份证件**，新生儿出生医学证明原件也可以。
- 一名持票成年人可以免费携带一名未满6周岁的儿童乘车，前提是儿童不单独占用座位。
- 不能只买大人的票后直接抱着宝宝上车，还需要提前办理“免费乘车儿童申报”。
- 免费乘车的宝宝没有单独座位，需要由大人抱着；如需单独占座，应购买儿童优惠票。
- 乘车当天建议携带大人身份证原件和宝宝户口本原件，并走人工核验通道。

## 免费乘车儿童申报步骤

### 方法一：铁路12306 APP申报

#### 第一步：添加宝宝为乘车人

打开铁路12306 APP，依次进入：

`我的 → 乘车人 → 添加`

按照页面填写宝宝本人的信息：

- 姓名；
- 证件类型；
- 证件号码；
- 出生日期；
- 联系方式；
- 优惠类型选择“儿童”。

宝宝已经落户的，可以使用户口本上的姓名和18位公民身份号码填写。证件信息必须与乘车当天携带的证件一致。

如果APP不支持所持证件类型、身份信息一直核验失败，或者无法完成添加，不要反复修改资料，直接带原件到车站人工售票窗口办理。

#### 第二步：购买成年人的车票

先正常购买同行成年人的车票。4个月宝宝选择免费乘车时，不需要单独购买有座车票。

#### 第三步：关联免费乘车儿童

成人票支付完成后，进入：

`火车票订单/订单 → 已支付`  
或  
`本人车票`

找到本次行程后：

1. 点击“添加免费乘车儿童”或页面顶部的“去添加”；
2. 选择携带宝宝的成年乘车人；
3. 选择刚刚添加的宝宝；
4. 点击“确认”，完成申报。

#### 第四步：检查是否申报成功

重新打开该笔订单，确认订单中已经显示宝宝的免费乘车信息，并核对：

- 宝宝姓名；
- 证件号码；
- 关联的成年人；
- 乘车日期、车次和到站。

免费乘车儿童会随关联的成人票一起改签；成人票退票后，关联的免费儿童信息也会一并取消。重新购票后需要再次检查申报状态。

### 方法二：人工售票窗口申报

如果已经买好成人票，但APP没有完成申报，可以在乘车前到车站人工售票窗口办理。

需要携带：

- 同行成年人的身份证原件；
- 宝宝的户口本原件，或出生医学证明原件；
- 已购买的成人车票订单信息。

向工作人员说明需要办理“免费乘车儿童申报”。建议比平时至少提前30分钟到站，节假日或客流高峰再多预留一些时间。

## 进站和乘车

1. 大人使用本人身份证进站。
2. 宝宝携带申报时使用的有效身份证件原件。
3. 户口本通常不能直接通过身份证闸机识读，建议主动走人工实名核验和人工检票通道。
4. 列车工作人员查验时，出示订单中的免费儿童信息和宝宝证件原件。

不要只携带户口本照片或复印件，以免现场无法完成实名核验。

## 宝宝是否需要买票

### 不单独占座

- 4个月宝宝可以免费乘车；
- 一名持票成年人限带一名免费儿童；
- 宝宝没有独立座位，需要大人抱着。

### 需要单独座位

- 使用宝宝本人有效身份证件购买儿童优惠票；
- 儿童票的日期、车次和席别应与同行成年人一致；
- 宝宝可以使用分配到的座位。

如果一名成年人同时携带两名未满6周岁的儿童，通常只有一名可以免费，另一名需要购买儿童优惠票。

## 出发前检查清单

- [ ] 成年人的高铁票已经购买；
- [ ] 宝宝已经添加为12306乘车人；
- [ ] 订单中已经完成免费乘车儿童申报；
- [ ] 宝宝姓名和证件号码填写正确；
- [ ] 携带成年人身份证原件；
- [ ] 携带宝宝户口本原件或出生医学证明原件；
- [ ] 提前到站并预留人工核验时间；
- [ ] 携带奶瓶、尿不湿、湿巾、备用衣物和薄毯。
